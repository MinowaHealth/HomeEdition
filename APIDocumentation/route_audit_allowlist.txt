# route_audit_allowlist.txt
#
# Paths that exist as Flask routes in UserApp/webapp/ but are intentionally
# NOT declared in openapi.yaml yet. The route_spec_audit.py audit treats
# these as "tolerated" — present, known, deferred — instead of new drift.
#
# Doctrine: openapi.yaml declares only stable contract surface. Routes
# whose request/response shapes are still evolving live here until they
# stabilize, then they migrate into the spec and get removed from this list.
#
# Groups below are organized by reason-for-deferral, not by file. When a
# group's whole row set has been declared in the spec, delete the group
# header AND its paths in one commit. The audit will fail with a STALE
# ALLOWLIST report if you forget.
#
# To add a new entry: add the path under the appropriate group with a
# brief reason. To remove: write the spec, then delete the entry.
